# BLCKey Clicker

> A game about compulsory completionism.

## Backstory

In Guild Wars 2, you can open these loot boxes called "Black Lion Chests." They mostly contain junk, but every month or so, the loot table gets a refresh. There is always one brand new item added, that can't be obtained anywhere else in the game, an "Exclusive." Over the years, I've done a few major research efforts into the droprates of these chests, and in February 2026, the game finally received an official disclosure of odds:

> Anet's official disclosure of odds: https://help.guildwars2.com/hc/en-us/articles/26106005230611-Disclosure-of-Odds

While I was working on a web-app simulator for opening BLC, I had a silly idea for an incremental game built around it, and BLCKey Clicker was born.

The alpha/preview version 0.1.0 released on April 1st 2026.
